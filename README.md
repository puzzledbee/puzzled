# puzzled
JavaFX logic problem solver

Puzzled helps resolve grid-type logic puzzles by parsing pseudo-english text clues and by performing analysis on the relationships.

Requirements:
* JDK 1.8
* commons-io-2.6
* controlsfx-8.40.14
