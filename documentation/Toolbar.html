Under the menu there will be a toolbar containing frequently-used buttons with graphic+text.FXML graphic tips:Using the CSS property to set the image is recommended over other approaches such as or an ImageView object inside the Button..png works best .svg did not, and .gif worked if loaded directly from the Internet but not from local file.In SceneBuilder, the button graphic icon will not show when pointing to a local file (relative file location not right). The icon will however show once the application launched. Place icons in the resources/icons/ directory (same level as src in NetBeans) and referred to as url("icons/new.png") provided the resources folder was added in NetBeans project properties-> sources.Showing the content of the running puzzled.jar file using unzip and watching for error messages loading the image file in the running output helps troubleshooting any issue.
