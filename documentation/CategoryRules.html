Type of text clue in which rules are specified between entire categories rather than between items.

Categories need to be considered by their numerical values. The category rules need to be parsed into one or more RestrictedValues. *e.g. The number of pulleys is never the same as the number of levers *e.g. The end weight is no less than 10 pounds fewer than the starting weight CategoryRules can be of the NormalClue type but also of the SpecialClue type, meaning the rule needs to be re-examined later on during the solving process. The problem PocketPuzzle Logic Puzzles 1 Unhappy New Year (problem 16) has both:

  • e.g. (implicit) none of the women sent a card to herself (NormalClue)
  • e.g. (in the header) none of the women sent a card to the same person from which she received a card (SpecialClue)
