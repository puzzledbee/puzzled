Logic Problem Clues. Simple text file of clues. Used mainly for testing of the processor instead of manually entering clues. It is convenient after resetting the problem, which clears the RelationshipTable, the Clue list, and Hint table. It is not necessary for normal operations as the .lpf or .lps files can already contain clues.
