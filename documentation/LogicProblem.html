Main class describing the problem data. It is serializable can be saved to / loaded from an XML .lpf file. Must contain as fields:

  • Problem title, source, notes
  • List of categories
    • Lists of items (within the Category objects)
  • Clues
  • Constraints

Relationships and HintsTable are not saved with the LogicProblem, but are instead regenerated from the constraints and clues.
