Default Relationship value, designates the association between two items is not known.

  • represented graphically by a blank GridCell.
  • its value is defined in the puzzled.data.Relationship class.

The relationship grid is initialized with all tiles worth VALUE_UNKNOWN at the onset, until their value is set by a Constraint, a Clue or a Deduction.

It is necessary to create all the relationships even before VALUE_YES or VALUE_NO is known because the relationships are also tiles in the User Interface on which the user can click GidCell.

See also VALUE_YES and VALUE_NO.
