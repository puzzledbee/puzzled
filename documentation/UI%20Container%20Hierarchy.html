Main UI container hierarchy (defined in FXML):

  • AnchorPane
    • NotificationPane (ControlsFX), used for messages
    • BorderPane (bPane)
      • TOP: VBox
        • Menubar
        • Toolbar
      • CENTER: main view
        • HiddenSidesPane (hsPane) (inserted at runtime, to bPane center region)
          • Content: TabPane (tPane)
            • Grid Tab Grid
            • Clue Tab
            • Answer Tab
          • Right: Label (pinLabel)
      • BOTTOM:
        • VBox
          • FlowPane (Clue text boox and text processing engine)
