UI component that extends StackPane which can be set to VALUE_YES, VALUE_NO or VALUE_UNKNOWN (private valueProperty) like its linked Relationship object.

  • it can represent a circle or a cross (the shapes' visible properties are bound to assertion matching one or the other of the valueProperty)
  • it has a context menu allowing the user to set a Constraint.

Setting the visible property of the primitives appear or add/removing the primitives appear equivalent.
