Logic Problem Shorthand. simple text file used to rapidly input problems into Puzzled, faster than manually generating an XML .lpf file or using the GUI to input all persistent [LogicProblem]] data. It is convenient for command-line operations and testing. See also .lpc.

.lps specifications

  • title followed by category names and item names for each category, preceded by a tab character
  • optional clues section follow. Clues will be distinguished from categories simply because no tab-led item follow
  • the constraints should be entered as clues
  • loaded .lps file become normal data structures which can then be saved as an XML-based .lpf
