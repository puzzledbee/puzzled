Clue numbering will follow the major.minor.sub convention. Generally, a given clue in a problem text may contain multiple sentences and/or sentences from which can be inferred multiple relationships.

  • major: matches the clue numbering in the text (starts from 1)
  • minor: matches the sentence number in the clue text (starts from 1)
  • sub: created if a single sentence includes multiple chunks from which multiple relationships can be inferred
