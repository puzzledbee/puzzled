Puzzled is an open-source java application which will ultimately help you solve grid-type logic problems by understanding pseudo-english translation of the clues provided with the problem.

This TiddlyWiki is used by the Puzzled developer to track requirements and features, decisions decision, algorithmic insight and other tidbits during the development of Puzzled.
