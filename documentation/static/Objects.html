A few main object: Puzzled -> main application

  • instantiates the logic processors
    • can create, load or save LogicProblems
  • loading a LogicProblem will follow the following steps:
    • de-serialize the .lpf
    • generates the Relationship data structure
    • assembles the Grid of GridCells
      • setting each relationship has the impact of drawing them on the Grid (if the ApplicationSettings says to do that)

LogicProblem -> main data structure:

  • can be saved and loaded
  • has categories (list/array?)
  • has items
  • has Clues and CategoryRules
  • has Constraints
  • does not have Relationships; they are re-computed upon loading
